# To Do List Manager
To do list tracker that displays tasks, assignee, and difficulty. Can toggle complete true/false and displays number of items pending.

## Name: Tyler Johnson

### Setup
Run `npm -i` to install dependencies listed in package.json.

#### ENV Requirements
N/A

#### Running the App
